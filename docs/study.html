Dataset

The base corpus is a public Reddit joke dataset (r/jokes dump), filtered to remove duplicates and empty entries. We retain 10,000 jokes along with their Reddit upvote score, used as a proxy for human approval in robustness checks but not in the main analysis.

Annotation procedure

Each joke was automatically assigned to one of nine bias categories — or labeled None — using Claude Haiku via the Anthropic Batch API. The classification prompt included 18 few-shot examples (2 per category + 1 "None") and an explicit definition of each category. Jokes targeting multiple groups were assigned multiple labels (e.g., "Racial/Ethnic/National; Gender-based").

To assess annotation quality, a researcher manually labeled five stratified samples of 100 jokes each (500 jokes total), enabling inter-rater reliability analysis between human and LLM annotations.

Methodology

3.1 Tournament Design

The experiment uses a round-robin tournament design: all 9 categories are paired exhaustively against each other (C(9,2) = 36 matchups), with joke position randomised across fights. This design ensures full coverage and allows position effects to be measured independently of category effects.

01
Sampling

154 jokes per category per matchup (the harmonic minimum across all categories). Fixed seed (42) ensures reproducibility.

02
Fight design

Each joke from category A fights 3 randomly drawn jokes from category B. Majority winner (2/3 rounds) wins the fight.

03
Position control

Joke positions are alternated across rounds to disentangle content preference from positional preference.

04
Judgment

Claude Sonnet receives both jokes and the prompt "Which of these two jokes is funnier?" with no additional context.

05
Aggregation

Results from both directions (A→B and B→A) are merged into canonical pairwise win rates. Global win rates aggregate across all opponents.

06
API scale

~80,000 individual requests submitted via the Anthropic Batch API. Total cost kept low through Claude Haiku for annotation and batching.

3.2 Confound Control: Joke Length

A natural confound is joke length: longer or more complex jokes may win simply because they contain more content, not because of their category. We compute mean and median character and word counts per category, and regress win rate on length. No significant relationship is found (r ≈ −0.2, p > 0.05; Kruskal-Wallis across categories: n.s.), ruling out length as an explanatory variable. The full length-control figure is on the Results page.

Statistical tests

Wilson 95% confidence intervals are computed for all win rates. Significance against the 50% null is assessed with two-tailed binomial tests. Per-matchup χ² tests assess distributional differences. The category × position interaction is tested with a χ² contingency table.
